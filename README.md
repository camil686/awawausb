## 🔍 Troubleshooting common issues

If you encounter trouble, use these steps to resolve common faults:

* **Device not detected:** Unplug the USB cable and plug it back into a different port on your computer. Ensure the device has power.
* **Browser connection fails:** Refresh your web page. Check that the awawausb icon appears in your system tray. If it does not appear, restart the application from your start menu.
* **Permission errors:** Your browser might block new devices by default. Check the address bar in your browser for a small icon that indicates blocked hardware access. Click that icon to allow the site to use the device. 
* **Slow response:** Close other programs that use USB ports to clear resources. Avoid using USB hubs if possible, as they sometimes interfere with data speed.
